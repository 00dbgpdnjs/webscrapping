웹 스크래핑 : 웹 페이지에서 내가 원하는 데이터 추출하는 행위
웹 크롤링 : 어떤 웹 페이지가 주어지면, 그 페이지 내에 있는 허용된 링크들을 따라가면서 모든 내용을 가져오는 것
tkinter's widget = html's element

xpath : 비슷한 엘리먼트가 있을 때, 어떤 엘리먼트를 지칭하는지 명확하게 하기 위해서 사용
        특정 엘리먼트를 지칭하는 주소
        - 표현방법 : 전체 경로  or  속성(like id or class)으로 더 간편하게

        ex) 학교 태그의 값이 "나도고등학교"이고, 학교 태그의 하위 태그로 학년 태그의 값이 1학년 이고, 하위 태그로 반 태그의 값이 1반이고, 하위 태그로 학생 태그의 값이 "1번" 부터 "5번"인 총 5개의 학생 태그가 있는데, 1번과 5번 학생의 이름이 같음.
        
            <학교 이름="나도고등학교>
                <학년 value="1학년"> ...
                
        xpath: 2번학생 :  /학교/학년/반/학생[2]  
            / : 현재위치에서 한 단계 아래
            -> solution: 학생 태그에 어트리뷰트로 학번으로 고유값을 줌  
                xpath:  //*[@학번="1-1-5"]  : ❗ KEY CODE ❗ 학번이 "1-1-5"인 녀석을 모든 문서[//*]에서 찾아주세요
                    // : 현재위치에서 모든 하위 엘레먼트에서 찾아봄
                    * : 모든 태그 ; * 대신 특정 태그명을 넣을 수 있음
                    @__ : __에 어트리뷰트
